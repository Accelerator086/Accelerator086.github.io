网上几篇不错的关于Markdown介绍 & Markdown语法：

  1. Daring Fireball Markdown Syntax
    这个是Markdown创始者之一的blog，介绍Markdown语法的Origin Documents
  2. Github: younghz/Markdown
    Github上一个关于Markdown的介绍，看不懂英文版的可以看这个，对应个人github blog为 Markdown
  3. 献给写作者的 Markdown 新手指南
    简书官方的Markdown介绍文档
  4. Learning-Markdown (Markdown 入门参考)

个人推荐Markdown写作平台：

  1. Typora Windows / Mac OS 等平台均可用，而且方便简洁，适合写作

  2. 印象笔记，马克飞象经常使用Evernote的人估计会很舒服，据说有道云笔记也支持Markdown语法，但不常用。

  3. Rmarkdown：在Rstudio里使用，配合R语言代码运行，适合做统计类小型报告（大型的就歇一会吧……），这个需要多加练习与使用才能掌握，如果掌握不了建议放弃，使用jupyter notebook敲。

为什么要用Markdown?

    因为Word写出来的学术报告特别难看，LaTeX写作对于新手来说上手比较难。简单来说就是这样。

    Markdown使用案例，和LaTeX的使用案例特别相似，或者在Word里使用公式编辑器后将所有公式转化为线性公式后也特别相似。

    这种公式敲出来的代码差不多算是囊括所有Markdown公式语法了。

    注意标号或者加粗或者其他的符号等，打完后记得带一个空格；大部分之间尽量多敲几行回车，要不然都堆在一块了……
