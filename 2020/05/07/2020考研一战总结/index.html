303 数学三 107分

因为自身学的是金融数学专业，学的数学课是数学分析（成绩：91、88、80）、高等代数（成绩：95、94）、概率论（成绩：91）、数理统计（成绩：84），所以按理来说应该不会有太大出入。

但是翻车了。

归根结底还是认识不到位。

考研数学应当着重练习计算能力，以及解题能力，而深入的数学更多侧重应用能力。

也就是说，以考研的标准来看，刷题没刷够。

一战使用的资料：

  1. 李永乐2020考研数学复习全书（全看完了）、历年真题解析（只按照关键的地方作了章节）、基础过关660题（一点也没做）、寒假作业系列（最后发现根本没有用）
  2. 张宇考研数学题源探析经典1000题（全做完了但没整理）
  3. 张宇八套卷（全做完了）、张宇四套卷（最后没完整做下来）
  4. 李林六套卷（做了一半）、李林四套卷（全做完了）

其实这么一看，数学做题明显少了，而且缺少很重要的一个步骤：整理反思。

错了的知识点并没有得到强烈重视。

二战的一轮学习安排：线性代数※→高等数学※→概率论与数理统计

二战的资料安排：

  1. 陈文灯2021考研数学复习指南（5月-6月，以此为主，至少2遍，啃透）
  2. 李永乐2021考研数学三复习全书（辅助）、历年真题解析（刷真题，专题系列每看完一科刷一科的专题，7月按照真题套卷再刷一遍2020-2010）、基础660题（要求做完，之后可以再刷2020年的作为补充，每看完一科就刷一科的题）
  3. 李永乐2021线性代数辅导讲义（看完线性代数后的补充+刷题，在第一阶段使用）
  4. 王式安2021概率论与数理统计辅导讲义（看完概率论与数理统计后的补充+刷题，在第一阶段使用）
  5. 张宇2021考研数学题源探析经典1000题（与2020年做对照，把错的地方重新做，并做好改错整理）
  6. 数学真题2020-2010（要求刷完至少两遍，不论怎样，7月一轮复习结束时一遍，9月二轮复习结束时第二遍，标记错题并整理）
  7. 李林六套卷、李林四套卷（【还没有购买】，最后冲刺使用）
  8. 合工大5套卷（2019、2020年版做就可以，不需要找2021年）

201 英语一 67分

按理来说，英语不应当这么低的分，但这么低的分其实也在情理之中。

自身英语水平从分数上来说，并不高也不低，四级（裸考）572分，六级（裸考）537分，IELTS 考了两次，第一次2017年12月14日 Overall 6.5（L6.5 R8 W6 S5），第二次2018年9月29日 Overall 7（L7 R8 W6.5 S5.5）。

看起来似乎英语至少不会弱，但我知道我英语怎样。

  1. 单词：单词量严重匮乏，这也是最大的短板，也是考研英语着重考察的短板。
  2. 语感，语法：很好，即使在不认识单词的情况下也知道文章在讲什么（XD）。
  3. 输出：较缺乏。

然后再加上英语真题只是做完一遍就不再看了（因为字迹问题，真的看不下去），所以GG其实也在情理之中。

英语一题型及分数分布：

  1. 完形填空。（20*0.5=10）
  2. 阅读理解。（4*5*2=40）
  3. 七选五新题型。（5*2=10）
  4. 翻译。（5*2=10）
  5. 作文。（10+20=30）

一般来说，作文至少扣10分，加上翻译扣分差不多接近15-20分，所以剩下的客观选择题最多只能扣5-10分，才有优势。

二战的资料安排：

  1. 英语真题2000-2008：文章通读一遍，精读完成，至少单词要全都过一遍（9*4=36，每天1篇都可以了，5-6月。）
  2. 英语真题2009-2020：做完一遍又一遍，所有地方都精读完成，做到能明白（12套，一周两套共6个周，7-8月）
  3. 英语真题 & 英语单词：Finish！
  4. 2020王江涛考研英语高分写作（一定一定要每个话题都要练习，在他的模板练会基础上，有自己的思考与见解/模板）

815 经济学综合 124分

得益于并不怎么良好的基础，以及对专业课的恐惧，还有今年分数特别松的情况，815专业课分数并不怎么低。

所以今年保持即可。

自身情况：微观经济学（平狄克课本）87分，宏观经济学（多恩布什课本）84分（期末85都没有拯救回来wdnmd）

平狄克：https://mitsloan.mit.edu/faculty/directory/robert-s-pindyck

多恩布什：http://economics.mit.edu/faculty/dornbusch/ （老爷子于2002年去世，但《宏观经济学》依然以三人的署名正式付印。）

《宏观经济学》多恩布什 网络学习资源：http://highered.mheducation.com/sites/0078021839/student_view0/index.html

一战复习情况：

  1. 微观经济学平狄克版 知识点过一遍+笔记整理，宏观经济学多恩布什版 & 高鸿业版 知识点过一遍+笔记整理
  2. 微观经济学平狄克版 课后题全部看答案做（chao）一遍，然后高鸿业版作了1.5h的补充
  3. 宏观经济学多恩布什版 & 高鸿业版 课后题全部看答案做（chao）一遍
  4. 815真题2014-2019按照专题做了整理，做完了2016、2017、2018三套卷的真题。

815 经济学综合 真题题型及分数分布：（其中微观经济学、宏观经济学各占一半）

  1. 选择题。（10*2=20）
  2. 判断题并解释原因。（6*4=24）
  3. 名词解释。（4*4=16）
  4. 计算题。（(10+15)*2=50）
  5. 论述题。（20*2=40）

要说不足也有，就是名词解释部分，该背的地方其实并不扎实，需要练习。

计算题还好，练的不算少，但也要继续练习。

大题幸亏没有出很难的论述题，而是以数学推导为主，不过今年需要强加练习。

101 思想政治理论 63分

首先政治不需要早期开始准备，尤其是论述题。

其次到最后的冲刺阶段，只看政治而且一定要背。所以政治的学习需要建立在其他科目的复习完全的基础上进行。

政治分为选择题+论述题。

选择题：政治知识点精讲精练+1000题训练+押题卷

论述题：押题卷。

一战使用资料：

  1. 肖秀荣考研政治：精讲精练（过知识点）、1000题（刷选择题，要求总结）、讲真题（其实不看也罢）、知识点提要（浓缩版的知识点）、形势与政策（补充形策部分）、考点预测（极浓缩的背诵材料）、肖八肖四（重中之重！！）
  2. 徐涛考研政治核心考案（其实是因为他的视频很好，不过才发现他的视频配合肖秀荣的书更有用）、形势与政策（说实话没用）、冲刺背诵笔记（也没用）
  3. 徐涛政治视频

二战也会继续如此，从8-9月开始。

The Final Summary

  1. 该考研时一定要好好备战考研，拒绝CS:GO，从现在做起。
  2. 该考研时一定要好好备战考研，单身的话请保持单身，不要恋爱，因为哪怕是谈恋爱，亦或是分手，都是影响的因素。
  3. 早睡早起是必要的，尤其是周末，因为考试是在周末进行的。上午 8:30-11:30 政治 + 数学三，下午 14:00-17:00 英语一 + 815经济学综合。一天的计划需要按照这个进行。
  4. 要下决心就要下的彻底，不可感动自己，不可半途而废。
  5. 二战相对于一战，最重要的是信心，要有能赢的勇气，要有实力的后盾。
  6. 执行力一定要有，做好计划后，计划好的事情一定要做到，而且尽量提前。给自己压力。
  7. 很想说一句“选择大于努力”（看中央财经大学的分数线，金融学院金融学362的分数线，而且一开始只说招15个人，2020年5月29日录取名单显示招25个人，更不用说中国经济与管理研究院、中国金融发展研究院的分数线了），但是毕竟还是想呆在小破猫里……

    尤其是要说一下国际关系学院，直接让人很糟心。

    这是我在2020年5月30日中午干刚好收到的，我也在短信上收到了相同的通知，然而过了没多久当我打开的时候，系统里就像“没事人”一样悄然撤销了复试通知……

    随后我打了国际关系学院研究生院（http://graduate.uir.cn/view?cid=5&tid=6）、国际经济系（http://ied.uir.cn/）的各个电话，然而在长时间没接的情况下，好几个“下一次”打过去居然是”您拨打的电话正在通话中“？？？

    wdnmd人傻了，真的。

    国际关系学院一生黑，哪怕它和国安局有联系dnmd。

    • 2020年6月2日：中共广东省委党校-西方经济学(020104) 复试通知下达，接受复试，后又主动放弃

      给中共广东省委党校点个赞，他们在发送复试通知前，首先打了个电话说明了一下情况（毕竟当天下午复试），然后我听到这么几个信息：

      • 我在调剂系统上的所有信息、学信网上的所有信息（本科院校等等），调剂单位都是能够看到的，这也是为什么他们在即使安排好复试的前提下，仍然接受了我的调剂。

      • 同上，所以我拒绝了中共北京市委党校的复试通知，是能看到的；国际关系学院的复试通知，也是能看到的（国际关系学院wgnmd）

      • 复试其实并不严格，只要按照教育部的要求准备必要的东西（身份证、准考证、承诺书签字），其他东西只要是“降维打击”，那其实并不太重要。

      复试负责老师立即和我联系，我觉得他们的服务挺不错的，可以稍微推荐一下（当然我是单纯因为研究院并不厉害，才不去面试的）

      那还是推荐一些其他院校的XDM考虑一下中共广东省委党校的调剂，负责经济学招生的老师很负责地给我打电话，询问是否继续参加面试。说实在的，弄得我很不好意思，就在这里强烈推荐吧。

结束了，就这样结束就好。
