在研究了100+优秀cv之后，我总结了简历写作的几个要点

发表于2020-02-03|更新于2021-02-23|Resume
|字数总计:1.1k|阅读时长:3分钟|阅读量:

  1. 一页纸简历。中文一页纸，英文也是一页纸，共两页纸。

    当然如果你特别优秀，你说一句话震惊整个业界那种，简历写多少页都没问题，详见金立群女儿金刻羽cv：http://personal.lse.ac.uk/jink/pdf/cv.pdf

  2. 简历风格一定要简约，走性冷淡风，不要放飞自我，详见这个回答的前半部分https://www.zhihu.com/question/20368865/answer/259314207

    而且，要发简历给对方，就给PDF版，别给word版。

  3. 简历必要元素：

    • 个人简介：简历上方最中央处，写上名字、电话、邮箱（可以用学校edu邮箱，别放qq邮箱、163邮箱，但是foxmail邮箱可以用，国内别用google邮箱）、地址+邮编（可以只写到你的学校，别写少了），你的标准一寸照照片（正式一点，比如穿西服去照，国内可以放照片，国外至少老美不让放，怕种族歧视。不可否认，照片上的你漂亮or帅气会给你的面试加分，但如果落差太大，就会减分）

    • 教育背景：你上的本科、硕士、博士是啥，我反正见过上的是np高中+优秀毕业生的经历也写在这里面了……

      • 主修专业，双学位专业/辅修专业，最终获得的学士/硕士学位
      • GPA or Average Score & Ranking
      • 交换经历（eg: Summer School）
      • （可选）修读主要课程+成绩

    • 获奖荣誉：获得的奖项，上面的所有时间内获得的荣誉，包括与专业方向相关或无关，能写就写，多了就列条，少了说具体情况，在有证明的前提下能吹就吹。【这部分可以放在对应时间的教育背景里按条来列，如果一页纸简历不够，或者确实少】

    • 个人经历：按照重要度排序：

      实习经历<>科研经历>比赛经历>>学生（课外）活动>其他经历

      • 实习经历：一般都是专业相关，Big Name + Relevant是最好的，若不兼得就看你自己取舍。正式实习生>PTA/远程（俗称“小黑工”）
      • 科研经历：发表论文>讨论班<>科研助手(<)>国创>科研立项>Extra，同样专业相关更好。
      • 比赛经历：可以写到“获奖荣誉”部分里。
      • 学生活动：如果是低年级可能还有用，高年级或者毕业了基本上就不看了，除非是“学生会会长”或同等水平。一点也没有完全可以。

    • 技能、证书、能力、其他：

      • 证书：能写就写，最能体现能力的硬指标。你说自己会计水平扎实，不如一句“CPA持证人”。如果只考出来几门，学生时代可以写“通过了几门”，毕业之后就只能写通过或未通过（未通过就不写了）。
      • （计算机）技能：可以列举，比如“Python, R, SAS, MATLAB”，但只能表明你接触过，不能表明精通（我cpp只会“#include”也写上去cpp了）
      • 语言：一般是英语，cet-4/cet-6，IELTS/TOEFL，GRE/GMAT（其实这个不算，但放在这里没问题）。当然你说你N1没有任何问题，不如说是加分项。
      • 活动：上面的“经历”比较水的可以放在这里列举，任职可以用括号后补充。
      • 兴趣爱好：文体啊啥的，随便写，证明你是一个interesting person。

  4. 简历拒绝的points：

    • 自我评价：谁管你怎么评价你自己。
    • 任何小错误都是不可以存在的
    • 在描述时候尽量使用客观语言（eg：运用matlab构建风险模型），别用主观性语言（eg：热爱祖国，擅长balabala）
    • 别用“乔布简历”https://cv.qiaobutang.com/
