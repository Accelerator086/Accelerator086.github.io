参考资料：（第一本为课本，第二本为第一本的深化与严格数学推导）

  1. 《An Introduction to Statistical Learning with Applications in R》(官方网址) ISLR
    • 这本书的中文版已经出版：《统计学习导论 基于 R 应用》
    • 笔记：
      • 统计学习导论公开课中文笔记 - 统计之都
        • 关于 ISLR 的相关网上资源整合
      • ISLR-python github.com
        • 《统计学习方法》 Python
      • ISL github.com
        • 《统计学习方法》 Python
      • RPubs - ppaquay
        • ISLR 课后题解答及 R 语言实现
      • asadoughi/stat-learning
        • Notes and exercise attempts for "An Introduction to Statistical Learning"
      • hardikkamboj/An-Introduction-to-Statistical-Learning
        • the exercises and its solution contained in the book An Introduction to Statistical Learning
  2. 《The Elementary of Statistical Learning : Data Mining, Inference, and Prediction》(官方网址) ESL
    • 这本书的中文版似乎并没有出版社接收，网上关于这本书的中文版对照如下：
      • Butters - 统计学习基础：译注
      • Wei Ya - ESL CN
  3. 《机器学习》周志华 清华大学出版社 （俗称“西瓜书”）
    • 笔记：
      • datawhalechina/pumpkin-book
        • 本书旨在对西瓜书里比较难理解的公式加以解析，以及对部分公式补充具体的推导细节。
  4. 《统计学习方法》李航 清华大学出版社
    • 笔记：
      • statistical-learning-method github.com
        • 《统计学习方法》笔记-基于Python算法实现
      • WenDesi/lihang_book_algorithm
        • 《统计学习方法》Python算法实现
      • SleepyBag/Statistical-Learning-Methods
        • 《统计学习方法》的硬核 Python 实现。
      • fengdu78/lihang-code
        • 《统计学习方法》的代码实现
  5. 《Machine Learning in Action》（官方网址）
    • 作者：Peter Harrington
    • 这本书的中文版已经出版：《机器学习实战》
  6. 《Hands On Machine Learning with Scikit Learn Keras and Tensorflow》
    • 作者：Aurélien Geron（Github：https://github.com/ageron）
    • 这本书的中文版为：《机器学习实战——基于Scikit Learn、Keras 和 Tensorflow》
